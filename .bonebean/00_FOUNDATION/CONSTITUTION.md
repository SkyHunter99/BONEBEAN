# Bone Bean AI Constitution

## Vision

Bone Bean AI is an AI-Powered Liquidity Provider Operating System.

## Principles

1. Engine must be deterministic.
2. AI must never execute trades.
3. Strategy is JSON.
4. Runtime executes compiled strategies only.
5. Simulation is mandatory before Runtime.
6. Every component must be pluggable.
7. Multi-tenant first.
8. Security first.
9. Event-driven architecture.
10. Domain-driven architecture.
11. Every execution must be explainable.
12. Every strategy must be versioned.
13. Every strategy must support migration.
14. User owns AI API Keys.
15. Platform owns Market APIs.
16. Revenue comes from Strategy Marketplace.
17. No hardcoded strategies.
18. No direct Engine-to-Engine dependency.
19. Worker-based execution.
20. Long-term maintainability over shortcuts.
