## Architecture Principles

- Modular by default.
- Explicit interfaces.
- Loose coupling.
- High cohesion.
- Plugin-first design.
- Event-driven where appropriate.
- Deterministic execution.

---

## Success Criteria

Architecture is complete when every module has:

- Architecture
- Interface
- Data Flow
- Event Flow
- Dependencies
