# Artificial Intelligence

The AI domain defines how Large Language Models integrate with Bone Bean AI.

Artificial Intelligence assists users by generating strategies, explaining decisions, optimizing parameters, and interacting through natural language. AI never performs deterministic trading logic or directly executes transactions.

---

- Strategy generation
- Strategy optimization
- Knowledge retrieval
- Prompt orchestration
- Memory management
- Natural language interaction

# AI Principles

- AI assists but never executes trades.
- AI cannot bypass deterministic engines.
- AI interacts through defined APIs only.
- Memory must remain tenant isolated.
- User data must never be shared across tenants.
- Every AI response should be explainable.
